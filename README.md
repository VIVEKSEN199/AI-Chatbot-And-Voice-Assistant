## ✨ Features

- 🎙️ *Voice Assistant* — Speak to the AI, get voice replies
- 💬 *AI Chat* — Powered by Groq's ultra-fast Llama 3.1
- ⚡ *Blazing Fast* — Groq API delivers instant responses
- 🎨 *Clean UI* — Minimal and responsive design
- 🔒 *Secure* — API keys stay local, never exposed
